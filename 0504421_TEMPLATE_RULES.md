# Правила разметки `0504421.ots`

Экспорт ODS умеет работать с редактируемым шаблоном `assets/0504421.ots`.

Если в `content.xml` шаблона есть маркеры вида `{{...}}`, приложение копирует OTS как основу, заменяет маркеры и сохраняет результат как `.ods`. Если маркеров нет, используется встроенная таблица.

## Общие поля

Эти маркеры можно вставлять в любые ячейки шаблона:

- `{{ORG_NAME}}` — наименование учреждения.
- `{{OKPO}}` — ОКПО учреждения.
- `{{STRUCTURAL_UNIT}}` — структурное подразделение.
- `{{RESPONSIBLE}}` — ответственный.
- `{{EXECUTOR_POSITION}}` — должность исполнителя.
- `{{EXECUTOR_POSITION_CODE}}` — код должности исполнителя.
- `{{DOC_ID}}` — ID документа табеля.
- `{{DOC_TITLE}}` — название документа.
- `{{DOC_DATE}}` — дата создания документа.
- `{{PERIOD}}` — месяц и год, например `Июнь 2026`.
- `{{YEAR}}` — год.
- `{{YEAR_SHORT}}` — две последние цифры года, например `26`.
- `{{MONTH}}` — номер месяца.
- `{{MONTH_NAME}}` — название месяца.
- `{{MONTH_LAST_DAY}}` — номер последнего дня месяца, например `30`.

## Заголовки дней

Для шапки календарных колонок:

- `{{DAY01}}` ... `{{DAY31}}` — номер дня, пусто если дня нет в месяце.
- `{{DAY01_MARK}}` ... `{{DAY31_MARK}}` — календарная отметка дня.

## Строки сотрудника

Создайте в шаблоне строку-образец сотрудника и вставьте в нее маркер `{{EMPLOYEE_NAME}}`.
Строка `<table:table-row>...</table:table-row>` с этим маркером будет продублирована по активным сотрудникам.
Если сразу после нее идет строка с маркерами часов (`{{D01_HOURS}}` ... `{{D31_HOURS}}` или `{{WORKED_HOURS}}`), эти две строки будут продублированы как единый блок сотрудника.

Доступные маркеры строк:

- `{{EMPLOYEE_NAME}}` — ФИО сотрудника.
- `{{DEPARTMENT}}` — подразделение.
- `{{PERSONNEL_NO}}` — табельный номер.
- `{{POSITION}}` — должность.
- `{{WORKED_DAYS}}` — отработано дней.
- `{{WORKED_HOURS}}` — отработано часов.

Для дневных ячеек строки сотрудника:

- `{{D01_CODE}}` ... `{{D31_CODE}}` — код дня.
- `{{D01_HOURS}}` ... `{{D31_HOURS}}` — часы.
- `{{D01_TEXT}}` ... `{{D31_TEXT}}` — код и часы одной строкой, например `Я 8`.

## Важно

- Данные берутся строго из сохраненных ячеек табеля. Если ячейка не сохранена, маркер дня будет заменен пустым значением.
- Для месяцев короче 31 дня маркеры лишних дней заменяются пустыми значениями.
- Форматирование строки-образца сохраняется из OTS-шаблона.
