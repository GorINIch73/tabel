# Табель 0504421

Десктопное приложение на C++20, CMake, SQLite и Dear ImGui для учета рабочего времени и подготовки данных табеля по форме 0504421.

## Сборка

```bash
cmake -S . -B build
cmake --build build -j 4
./build/tabel0504421
```

При первой конфигурации CMake скачивает Dear ImGui docking branch и ImGuiFileDialog через `FetchContent`.

## Реализованный стартовый срез

- открытие и создание SQLite-баз через ImGuiFileDialog;
- автозагрузка последней базы из пользовательских настроек;
- SQLite-схема для учреждения, сотрудников, видов активности, норм времени, календаря и ячеек табеля;
- отдельный справочник физлиц с личными данными;
- отдельный справочник должностей;
- справочник сотрудников как кадровых записей: физлицо, должность из справочника, подразделение, ставка, табельный номер;
- стартовые виды активности: работа, выходной, основной отпуск, отпуск ЧАЭС, больничный, за свой счет;
- документы-периоды активности для отпусков, больничных и других отклонений;
- перезаполнение табеля: рабочий день `Я` 8.2 часа, выходные по календарю, затем перекрытие документами-периодами;
- стартовый дашборд с быстрыми показателями и ближайшими отпусками, больничными и другими периодами;
- справочники в формате “список + редактор в том же окне” с автосохранением;
- табельная сетка по сотрудникам и дням месяца;
- выбор кода дня из справочника активности;
- производственный календарь: локальная генерация выходных и загрузка календаря РФ с `xmlcalendar.ru`;
- ручная корректировка календарных дней;
- темы, настройки, окно “О программе”;
- докинг-интерфейс Dear ImGui на всю рабочую область, закрываемые окна, активный splitter, фильтруемый combo по ФИО.

## Ближайшие доработки

- печатная форма 0504421 и экспорт в ODS/XLSX/PDF;
- нормы часов с расчетом неполной ставки по сотруднику;
- массовое заполнение периода видом активности;
- удаление/архивация документов и справочников;
- контроль пересечений периодов и итоговые суммы по строкам табеля.
